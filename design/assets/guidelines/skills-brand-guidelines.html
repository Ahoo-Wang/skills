01. 品牌战略 / Brand Essence

一句话定位

skills 是面向 AI Agent 的开源技能聚合枢纽，将多个源仓库的技能以 source-owned plugin 的方式同步、拆分并分发。

核心关键词

MModular 模块化
每个 Skill 与 Plugin 都可按源项目独立组织。
OOpen 开放
围绕开源协作、可追踪元数据与透明同步构建。
{ }Developer-first 开发者优先
文档、命令、徽章、插件卡片优先服务工程使用场景。
↻Automated 自动化
用 sync、flow、clock 等视觉符号强化“持续同步”。
✓Reliable 可靠
使用深色基调、明确边框、清晰层级体现工程可信度。
+Extensible 可扩展
插件瓦片与节点网络可无限延展到更多源项目。

品牌叙事

从“单个技能文件”升级为“技能生态入口”：标志中的连接节点代表技能源、插件、Agent 工作流和自动化同步之间的连接关系。

受众

开源维护者、AI Agent 用户、Claude Code / Codex 插件使用者、架构师与全栈工程团队。

语气

精确、直接、帮助型、工程级。避免营销夸张，不使用虚构指标或无法验证的能力宣称。

视觉原则

多用网格、圆角卡片、流程线、状态徽章和 CLI 片段；少用复杂插画、过度霓虹或过度拟物。

02. 标志系统 / Logo System

Logo Concept

一个由连接 skill blocks 构成的模块化 “S”。上、中、下三个节点分别象征 source、plugin 与 agent flow；渐变表达跨仓库同步与知识流动。

Logo Suite

Primary logo
Icon only
Monochrome

使用规则

全彩标志用于白色或 Cloud 背景；单色标志用于低色彩、印刷、代码文档或深浅对比受限的场景。

数字端最小尺寸：图标 24px；完整字标宽度 120px。标志四周至少保留 1 个节点半径的留白。

不要这样使用

不要改变渐变方向或替换为随机颜色。

不要压扁、拉伸、旋转或加复杂描边。

不要置于高噪声背景或低对比图像上。

03. 色彩与 Token / Color System

Midnight#0B1020

Primary text, monochrome logo, terminal surfaces

Electric Blue#346CFF

Primary brand action, section headings, sync accents

Cyan#39D0FF

Flow lines, code/API accents, secondary highlights

Mint#73F0C0

Success states, active plugin nodes, positive completion

Cloud#F5F7FB

Background surfaces, cards, panels

Slate#6B7280

Secondary copy, captions, metadata

比例建议

70% Cloud / White 用于背景与留白；20% Midnight / Slate 用于文本和结构；10% Electric Blue / Cyan / Mint 用于强调、状态与连接线。

状态映射

Electric Blue = primary/action/sync；Cyan = API/code/flow；Mint = success/ready/agent active；Midnight = terminal/serious context。

05. 应用规范 / Applications

README / GitHub

顶部使用完整 logo 与一句话说明。徽章保持事实性，例如 Apache-2.0、sync 6h、source-owned plugins。避免虚构 star、下载量或性能指标。

插件市场 / Plugin Marketplace

插件卡片优先显示 plugin name、source project、简短功能描述、标签与安装命令。

CLI / Docs

命令行片段使用 Midnight 背景，Cyan 标示命令参数，Mint 标示成功状态。

CLI 示例

$ skills
$ plugin add ahoo-wow-skills
Resolving plugin... ✓
Fetching from Wow... ✓
Installing... ✓
Done. Ready to use.

文案原则

短句、明确、可验证。以“如何使用”和“状态反馈”为主，避免宏大但不可验证的口号。

可访问性

正文优先使用 Midnight on White / Cloud；Slate 仅用于辅助文本。蓝色小字不要直接放在 Cyan 或 Mint 上。交互状态除了颜色，还应使用图标、文字或形状差异。
