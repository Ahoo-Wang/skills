# skills 品牌资产包 v1

本资产包为 Ahoo-Wang/skills 开源项目定制，用于表达“Open Source Agent Skills Hub”的品牌方向：模块化、开放、开发者优先、自动同步、可靠、可扩展。

## 目录

- `guidelines/skills-brand-guidelines.pdf` - 中文品牌设计规范指南
- `guidelines/skills-brand-guidelines.html` - 可编辑 HTML 源文件
- `brand-board/skills-brand-board.png` - 确认版 Brand Board
- `logo/svg/` - SVG 标志资产，可用于网页、README、文档和设计工具
- `logo/png/` - 透明 PNG 导出，含 512/1024/2048 等尺寸
- `tokens/` - CSS、SCSS、Tailwind 与 JSON 设计 token
- `examples/` - 插件卡片、CLI 片段、仓库徽章的 SVG/PNG 示例
- `patterns/` - 流线与插件瓦片图形语言素材

## 推荐用法

1. README 或网站首页使用 `logo/svg/skills-logo-primary.svg`。
2. favicon、头像或小尺寸场景使用 `logo/svg/skills-mark.svg`。
3. 单色、印刷或低色彩环境使用 `logo/svg/skills-logo-mono.svg`。
4. 产品 UI 使用 `tokens/skills-brand.css` 中的 CSS 变量。
5. 不要拉伸、旋转、换色或在高噪声背景上直接使用全彩标志。
